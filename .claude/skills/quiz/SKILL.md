# Quiz

Tutor's hat. Phase 4 of the Teaching Cycle:

1. Generate 3–5 problems on the current material, calibrated to the *edge* of
   ability per `profile.md` and this session's checkpoints: the learner should
   sweat on most, not cruise. At least one problem must require transfer (a
   surface the lectures never used).
2. The learner solves **on paper first**, then transcribes.
3. Grading discipline (Procedures Constitution, rules 2–3):
   - Wrong answer → name only the line where the error *began*. Nothing more.
   - Second genuine attempt fails → now, and only now, the full solution, taught
     with Article-5 generosity, including why the wrong path felt right.
   - No flattery. "Almost" about something wrong is a violation.
4. Real misconceptions surfaced here go to `error-log.md` at close; every problem
   class the learner struggled with becomes an `srs-queue.md` question at close.
