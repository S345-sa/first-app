# Exemplars — The Gold Standard Shelf

The Critic (Forge, pass 2) judges every draft against the passages in this folder.
A model in front of the machine pulls it toward the masterworks' region far more
strongly than any description of them.

## Why this folder starts empty — the honest reason

Short quotation with attribution for study and criticism is lawful in most
copyright regimes (Berne Convention, art. 10). That is not the obstacle. The
obstacle is **fidelity**: a predictive model does not store these books verbatim
and cannot retrieve exact passages — asked to "quote Feynman," it would produce
convincing near-Feynman with silent errors, falsely attributed. In the one folder
whose entire job is *exactness*, a plausible fake is worse than an empty shelf.

So the division of labor is strict:
- **The learner** supplies the verbatim excerpts, copied from copies they own,
  for private study in this private repository. Keep each excerpt short
  (1–3 pages): the *anatomy* matters, not the bulk.
- **The machine** supplies what it genuinely can: original anatomies of these
  passages' methods (`anatomies.md`) and original specimens written to the same
  standard (`specimen-*.md`) — judged, never trusted.

## Anatomy notes (do this once per passage — it doubles the folder's power)

*what exactly* this passage does that ordinary textbooks don't. The Critic quotes
these anatomies back at drafts that fall short.
