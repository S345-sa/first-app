# Error Log — The Chapter No Printed Book Ever Had

> Every real misconception the learner forms, in the exact three-part format below.
> Appended at the end of every session. This file becomes the book's final chapter:
> a map of this specific mind's traps, and how each was escaped.
>
> Rules:
> - Only REAL misconceptions (a wrong belief that felt right) — not typos or slips.
> - "Why it felt right" must be written with respect: the error's internal logic,
>   honestly reconstructed. An error that felt random teaches nothing.
> - "The correction" is written in the learner's own final words, not the tutor's.
